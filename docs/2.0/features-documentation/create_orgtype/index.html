Organization Type Creation

Contribution Guidelines    Edit | Report

Contents

[hide]
  • 1 . Prerequisites
  • 2 . Creating Organization Types
  • 3 . Updating Organization Type

On Sunbird there are different types of users. However, the roles and responsibility of each user varies according to the role they are assigned on the portal. For example, the responsibility of a course creator is to create a course while that of a course reviewer is to review a created course etc. Similarly, a system administrator on Sunbird has specified roles.

On Sunbird, only the system administrator has the right to add organization types on the portal. Once these organization types are added, the organization administrators can upload other organizations and users. Organization type can vary according to requirements. Any number of organization type can be added on Sunbird.

Prerequisites

Step Screen
1. You are logged in as the system administrator
2. You are currently on Profile page. You want to create organization types

Creating Organization Types

Step Screen
1. Click Setup from the profile drop-down menu
The Org Types page is displayed
1. Click +icon on the right side of the screen. The Add Org Type page is displayed
2. Enter the type of organization you want to create
3. Click Add to add the organization Type
4. Click Cancel to go back to the Org Types screen

Updating Organization Type

Step Screen
1. All organization types created are listed in the Org Types page
2. Click Edit icon to edit the created organization type
1. Edit the name of the organization type
2. Click Update to save the edited changes
3. Click Cancel to go back to the Org Type page
