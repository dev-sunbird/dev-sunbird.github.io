Overview

Sunbird Mobile App offers extensive functions and features, which allows users to perform various actions:

  • Sign in with registered credentials
  • Consume the app using Guest account
  • View content, Online as well as Offline
  • Search content from the Sunbird portal online
  • Search downloaded content from the mobile device when offline
  • Scan QR code or Dial code in textbooks and access related content
  • Download content, when connected to the Internet
  • Enrol for available courses
  • Take up assigned courses
  • Browse and view resources from the content library
  • Create and update user profile
  • Share content with other users
  • View and rate the content

Downloading and Installing Sunbird Mobile APP

Prerequisites

  • You have Android OS 5.0 and above on your device
  • You have Github URL for installing Sunbird Mobile App
Step Screen
You can directly install Sunbird Mobile app using this URL
1. Click staging.apk link to install latest mobile built
2. Click Download to download the app. You will be redirected installation screen.
3. Click Install to install the mobile application
1. Click Allow to provide access to the following data:
  • Access Photo, Media and files
  • Take picture and record videos
  • Access device's location
  • Make and manage phone calls
    • 2. Select your preferred language
    3. Click Continue to start navigating through the app
    Note: Denying these permissions diables users from downloading content for offline use

    Signing In

    Step Screen
    1. Tap the Sign In to log in into Sunbird Mobile App. You will be redirected to Login screen
    2. You can Log In to Sunbird Mobile App using either of the three options: registered Username or Email ID or Phone number
    3. Enter your Password
    4. Tap Login button
    Note: You can also sign in to the application without using your credentials. For more details refer Browse as Guest

    Exploring Sunbird Features

    Sign in to Sunbird Mobile App with your registered credential. You will land to Home screen. Sunbird Mobile app allows you to navigate to the different features. Tap associated icons to navigate to the relevant screen of the app.

    Home

    Step Screen
    1. Swipe to view all the courses that you are enrolled in To Do or click View all
    2. Click Resume to take up the a course
    3. Click View all to view Announcements
    3. Tap the QR Code icon to scan QR codes or Dial codes in physical textbooks
    4. Tap Search icon. You will be redirected to next screen
    5. Type keywords using your mobile keypad to search for content

    Courses

    Step Screen
    The Courses page lists your current courses and their progress, latest course offerings and popular courses
    1. Swipe the associated course category to browse the relevant courses in that category. Courses are categorized as:
    a) Courses In Progress
    b) Latest Courses
    c) Popular Courses
    2. Tap Search or Filter icon and enter keywords to search for relevant courses
    3. You can filter the courses by: Board, Subject and Medium
    4. Click Apply to apply the filter
    5. Click Cancel to exit the screen
    1. Click Resume to take a course
    2. You will be re-directed to next screen. Click Start to resume your course
    3. To download the course click Download all
    Note: After the content is downloaded, you can view the content offline
    1. Click Open to view available course
    2. Click Enroll in Course to get enrolled for the course having a batch
    1. Click Share icon to share the content As Link or As File
    2. Click on Menu icon to Delete or Report course
    Note: You can select reasons for reporting content viz,
  • Inappropriate content
  • Copyright violation
  • Data privacy violation
  • Other reason

    1. Tap on QR Code Scanner icon to scan QR Codes or Dial Codes

    Browse as Guest

    You can browse the Subird Mobile App without sining in. Use Browse as Guest feature to navigate through the app.

    Step Screen
    1. Tap on Browse as Guest
    2. Select your guest mode as Teacher or Student
    3. Click Continue to enter into the app
    1. You will be re-directed to Home screen
    2. Click Select Board to search and filter your content by selecting a particular board
    3. Select a Board
    4. Click Save to save your seection
    5. Click Cancel to exit the screen

    Note:

    1. Following features are not available for users accessing the application on Guest mode:
    • Enrolling to a Course
    • Limited access to edit or update Profile. Users can edit or update Name, Grade, Subjects and Medium of Instruction
    • Uploading of Profile Image
    1. Users, on Guest mode, can access Course and Library same as registered user
