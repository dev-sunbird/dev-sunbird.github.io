One of the many responsibilities of the organization administrator on Sunbird is to assign roles and responsibilities to users within an organization. The administrator can assign more than one role to a user.

Prerequisites

Step Screen
1. You have organization administrator permissions
2. You are logged in as an administrator
3. You are currently on Profile page. You want to assign roles to different users in your organization

Select User

Step Screen
1. Select Users from the drop-down menu beside search bar
2. Enter name of the user
3. Press Enter or click the Search Icon

View User Search Results

Step Screen
1. Click Search to view the user details list
2. For any registered user on Sunbird, you can:
 a) Assign Badges
 b) Edit User Role
 c) Delete User
3. Click on the user name to view the profile details of the user
4. Click Download to download details of all users uploaded. The details get downloaded in an excel file
5. Click Close icon to exit the search results display

User Profile

Step Screen
1. On clicking the user name, the following profile details can be viewed:
 a) Summary
 b)Address
 c) Experience
 d) Education
 e) Skill Tags
 f) Basic Profile
2. Click the close icon to go back to the list of users page

Awarding Badges

Step Screen
1. Click the medal icon to award a badge to the user
1. Select Yes to award a badge
Note: *
 a) Badge once awarded cannot be revoked
 b) Badges can be assigned only once

Assigning User Role

Step Screen
1. Click on the Edit icon to assign roles to the user. The Select Role page is displayed
1. Names of different organizations that have been uploaded are available here. Select the organization, of which users must be assigned roles, from the drop-down
2. To select and assign user role, click on the check box. To un-assign the role, click on the checkbox to clear it.
Note: A user can be assigned more than one roles
3. Click Update to save the assigned roles
4. Click the close icon to go back to the list of users page

Blocking Users

Step Screen
1. Click on the delete icon to delete a user
2.Click Yes on the Block User dialog box to delete the user
Note: To unblock a user, user details have to be uploaded again. To upload user, refer Register Users
