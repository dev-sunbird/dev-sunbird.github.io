User Profile

Contribution Guidelines    Edit | Report

Creating and updating your profile on the platform gives you a unique identity on Sunbird. This helps the system identify your skill set, experience, location, etc. Providing this information helps the system provide personalized recommendations for courses to upgrade your skills, selects you for custom programs, connects you to others in the community, build bridges for collaboration, etc. You may choose to create, update or edit your profile information at any time.

To create or update your profile:

Step Screen
1. Login with the registered credential, to land into Home page
2. Click Profile tab on the header
Or
3. Click Profile icon to the right
1. Depending on your organization, you may see some information are already available, viz: Username, City you belong to etc
2. Click on Edit icon on your profile, to add profile image
3. You can also see your Last Login Time

Note: Maximum image upload size is 4 MB
Organization setup option:
1. Click Search dropdown button
2. Select Organizations from the dropdown list
3. Click Filter icon
4.Select Organization Type from the dropdown list
5. Click Apply button to associate an organization with the organization type
6. Click Reset button to select attributes again
Filter Profile by attributes:
1. Click Search dropdown button
2. Select Users from the dropdown list
3. Click Filter icon
4. You can filter the profile by:
  • Grades
  • Medium
  • Subjects
  • Roles
  • Location

    2. Click on Plus icon to add details viz, Profile Description, Language known, Profile picture

    Education: Add or edit your academic qualifications, such as degree, year of passing, percentage, board/university, etc. You can show or hide your academic details from others
    Skill Tags:
    1. Click the Add to open Add Skills page
    2. You can show or hide your skill details from others
    3. Enter or select skills from the drop down
    4. Click Finish to save the details
    5. Click Cancel to exit the page
    Skill Endorsement:
    1. Click View More dropdown in Skill Tags section
    2. Click + icon to endorse for a particular skill
    3. Click View Less to collapse the Skill Tags page
    Additional Information:
    1. Add or edit personal information such as name, phone number, date of birth, location, languages known, etc
    2. Add your Social Media Links such as Facebook, Twitter, LinkedIn etc
    3. Click Save button to save the changes
